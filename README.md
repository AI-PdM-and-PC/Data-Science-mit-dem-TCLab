# Data Science mit dem TCLab

This repository contains German teaching materials for data science with the Temperature Control Lab (TCLab). The notebooks are based on the APMonitor Data Science with the TCLab course and were revised for teaching in process automation, applied AI, and process-related data analysis.

The material is intended as an introductory, practice-oriented notebook collection. It focuses on working with experimental and simulated process data, preparing data for analysis, visualizing time series, and applying basic data science methods in a technical context.

## Target audience

The material is intended for students in process automation, applied AI, data science, and predictive maintenance.

The notebooks are suitable for guided teaching, lab sessions, and self-study. Basic Python and Jupyter knowledge is assumed.

## Contents

The repository covers introductory data science topics with a focus on the TCLab and process-related time series:

- Python and Jupyter basics for data analysis
- importing and preparing measurement data
- exploratory data analysis
- visualization of process data
- interpolation and missing values
- time-series analysis
- TCLab-related examples
- preparation of data for later modeling and monitoring tasks
- first steps with feature engineering, classification, regression

## Repository context

The Temperature Control Lab is a small experimental setup for teaching process dynamics, data analysis, and control. It provides temperature measurements and heater signals that can be used to study typical questions in process automation.
